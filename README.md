Student File Handling Project (Python)

A simple Python mini-project for writing and reading student data using file handling.
Perfect for 2nd-year engineering students and GitHub portfolio projects.

Project Overview

This project demonstrates:

How to write student data into a text file

How to read the stored data

Basic usage of Python file handling (open(), write(), read(), with statement)

The program stores student roll number, name, and marks, and then displays the saved data on the screen.
